compass.validation.content.LegalTextValidator#

class LegalTextValidator(tech, *args, score_threshold=0.8, doc_is_from_ocr=False, **kwargs)[source]#

Bases: StructuredLLMCaller

Parse chunks to determine if they contain legal text

Parameters:

  • tech (str) – Technology of interest (e.g. “solar”, “wind”, etc). This is used to set up some document validation decision trees.

  • score_threshold (float, optional) – Minimum fraction of text chunks that have to pass the legal check for the whole document to be considered legal text. By default, 0.8.

  • *args – Parameters to pass to the StructuredLLMCaller initializer.

  • **kwargs – Parameters to pass to the StructuredLLMCaller initializer.
